# GitHub About Settings

GitHub's right-side About panel is repository metadata, not a file in the repo.
Use the gear icon next to **About** on the repository page, or open repository
settings, then paste these values.

## Notes

- Forking your own repository is disabled on GitHub. The fork button is useful
  for other users viewing the repository from their own accounts.
- If forking is disabled for everyone else too, check **Settings > General >
  Features > Allow forking**.
